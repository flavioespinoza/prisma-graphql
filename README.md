## Getting Started
Clone the repo
```bash
git clone https://github.com/flavioespinoza/prisma-graphql.git
```

Use `yarn` to install dependencies,
```bash
yarn install
```

## Prisma
Deploy Prisma GraphQL Api
```bash
prisma deploy
```

Generate CRUD building blocks
```bash
bash crud.bash
```

Playground
- http://localhost:4466/

Admin
- http://localhost:4466/_admin

## Data
Generate user
```bash
yarn start
```
